Simple Currency Converter App :

1:- Description :

The Currency Converter App is a web application that allows users to quickly and easily convert between different currencies. The app is built using Vue.js CDN and utilizes switch case statements to handle the different conversion scenarios.

2:- Features :

User-friendly interface with input fields and select dropdowns for currency to convert from, and currency to convert to. Conversion logic implemented using switch case statements to handle different currency conversion scenarios.

3:- Technologies Used :

HTML, CSS, Vue.js CDN
