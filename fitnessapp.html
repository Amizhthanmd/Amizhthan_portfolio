Fitness App :

1:- Project Description :

The Fitness App is a mobile application that allows users to track their fitness goals and maintain a healthy lifestyle. The app provides various features that help users to calculate their body mass index (BMI), protein intake, and water intake. Additionally, it includes a range of exercises that users can perform to achieve their fitness goals.

2:- Project Overview :

The app is built using Vue.js, a progressive JavaScript framework for building user interfaces, and Vuetify UI, a material design component framework for Vue.js. The backend is developed using Golang, a fast and efficient programming language designed for building reliable and scalable applications.

3:- Features :

(1) The BMI calculator allows users to calculate their BMI by entering their height and weight. The app then displays their BMI along with a chart that shows where their BMI falls on the scale.

(2) The protein calculator helps users to determine their protein intake based on their body weight and fitness goals.

(3) The water intake calculator allows users to calculate their daily water intake based on their weight.

(4) Users can select from a range of workouts, including Chest workouts, Biceps workouts, Triceps workouts, Back workouts, Shoulder workouts and Forearm workouts.

4:- Technologies Used :

(1) Backend: Golang

(2) Frontend: Vuejs, Vuetify
